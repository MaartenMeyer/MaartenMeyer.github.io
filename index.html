Over dit portfolio

Dit portfolio is opgesteld in april 2020 en bevat ten enkele grotere projecten die ik voor mijn studie informatica heb gemaakt. De meeste van deze projecten zijn in groepsverband gemaakt en slechts een klein deel is individueel ontwikkeld. Van elk project geef ik in dit portfolio een korte beschrijving van het project met daarbij mijn eigen bijdrage aan dat project.

Naast de projecten die in het kader van de studie zijn uitgevoerd heb ik ook aan een aantal persoonlijke projecten gewerkt vanwege bepaalde interesses of om extra kennis op te doen over een bepaald framework. Buiten studie om ben ik vooral veel bezig geweest met JavaScript (Node.js), Angular en Vue.js.

Vanwege de eenvoud van dit portfolio heb ik gebruik gemaakt van een ontwerp van HTML5 UP. In de gemaakte projecten is alle gebruikte HTML en CSS code wel zelf geschreven.
